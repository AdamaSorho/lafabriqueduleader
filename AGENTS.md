# Project Conventions

## Brand Naming

- The organization's canonical name is `Zonzerigue Leadership International (ZLI)`.
- Never prefix the organization name or `ZLI` with `DS`.
- Always write `Zonzerigue` without an accent, including when referring to the author.
- Use `DS Zonzerigue` only for the person, never for the organization.
- Before completing brand or content changes, verify that neither `ZonzeriguĂ©` nor `DS Zonzerigue Leadership International` remains in user-facing copy.

## AWS

- Use the AWS CLI profile `TerraformMindapax` for this project's AWS operations.
- The deployed Lambda and Terraform infrastructure are in `us-east-1` unless the repository configuration says otherwise.
